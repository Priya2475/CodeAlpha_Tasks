# 📉 Unemployment Analysis with Python

## 📌 Project Overview
This project analyzes unemployment data in India using Python. It includes data cleaning, visualization, and trend analysis to understand unemployment patterns and the impact of COVID-19.

## 🎯 Objectives
- Analyze unemployment trends.
- Perform data cleaning.
- Visualize unemployment rates.
- Study COVID-19 impact.
- Identify seasonal patterns.

## 📂 Dataset
- Unemployment in India Dataset

## 🛠️ Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn

## 📊 Analysis Performed
- Data Cleaning
- Exploratory Data Analysis (EDA)
- Monthly Trend Analysis
- State-wise Unemployment Analysis
- Correlation Analysis

## 📈 Results
The analysis shows a significant increase in unemployment during the COVID-19 lockdown period, along with noticeable regional and seasonal variations.

## 🚀 Learning Outcomes
- Data cleaning
- Data visualization
- Exploratory Data Analysis
- Trend analysis
- Insight generation
