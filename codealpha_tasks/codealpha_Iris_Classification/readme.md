# 🌸 Iris Flower Classification

## 📌 Project Overview
This project builds a Machine Learning classification model to identify the species of Iris flowers based on their sepal and petal measurements. The model classifies flowers into one of three species: Setosa, Versicolor, or Virginica.

## 🎯 Objectives
- Load and preprocess the Iris dataset.
- Train a classification model.
- Predict flower species.
- Evaluate model performance.

## 📂 Dataset
- Iris Dataset
- Features:
  - Sepal Length
  - Sepal Width
  - Petal Length
  - Petal Width
- Target:
  - Setosa
  - Versicolor
  - Virginica

## 🛠️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib

## 🤖 Machine Learning Algorithm
- K-Nearest Neighbors (KNN)

## 📊 Evaluation Metrics
- Accuracy Score
- Confusion Matrix
- Classification Report

## 📈 Results
The KNN model achieved high classification accuracy and successfully predicted the flower species.

## 🚀 Learning Outcomes
- Data preprocessing
- Classification using Machine Learning
- Model evaluation
- Performance analysis
