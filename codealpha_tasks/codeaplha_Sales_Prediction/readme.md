# 📊 Sales Prediction using Python

## 📌 Project Overview
This project predicts product sales based on advertising expenditures across different media channels using Linear Regression.

## 🎯 Objectives
- Analyze advertising data.
- Train a regression model.
- Predict sales.
- Evaluate model performance.
- Generate business insights.

## 📂 Dataset
- Advertising Dataset

### Features
- TV Advertising
- Radio Advertising
- Newspaper Advertising

### Target
- Sales

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🤖 Machine Learning Algorithm
- Linear Regression

## 📊 Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squed Error (RMSE)

## 📈 Results
The Linear Regression model successfully predicted product sales, showing that TV and Radio advertising have a stronger impact on sales than Newspaper advertising.

## 🚀 Learning Outcomes
- Regression analysis
- Sales forecasting
- Business analytics
- Model evaluation
- Data visualization
