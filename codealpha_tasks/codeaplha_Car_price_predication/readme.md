# 🚗 Car Price Prediction using Machine Learning

## 📌 Project Overview
This project predicts the selling price of used cars using Machine Learning regression techniques based on vehicle characteristics.

## 🎯 Objectives
- Load and preprocess the dataset.
- Perform feature engineering.
- Train a regression model.
- Predict car prices.
- Evaluate model performance.

## 📂 Dataset
- Car Price Dataset

### Features
- Year
- Present Price
- Driven Kilometers
- Fuel Type
- Selling Type
- Transmission
- Owner

### Target
- Selling Price

## 🛠️ Technologies Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## 🤖 Machine Learning Algorithm
- Random Forest Regressor

## 📊 Evaluation Metrics
- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)

## 📈 Results
The Random Forest model accurately predicted car prices and achieved a high R² score.

## 🚀 Learning Outcomes
- Feature engineering
- Regression models
- Model evaluation
- Price prediction
